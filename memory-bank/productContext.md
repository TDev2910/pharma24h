# Product Context - SUCKHOE24H (KiotViet-Inspired)

## 🎯 BUSINESS CONTEXT

### Project Overview
**SUCKHOE24H** là một hệ thống quản lý dược phẩm và chăm sóc sức khỏe toàn diện, được thiết kế dựa trên các mô hình thành công của [KiotViet](https://banhdacua123.kiotviet.vn/man/#/DashBoard) - một trong những nền tảng quản lý bán hàng hàng đầu tại Việt Nam.

### Core Business Value
- **Healthcare Management**: Quản lý thuốc, dịch vụ y tế, và chăm sóc sức khỏe
- **Inventory Control**: Theo dõi tồn kho real-time với báo cáo chi tiết
- **Multi-category Products**: Hỗ trợ 4 loại sản phẩm chính (Thuốc, Hàng hóa, Dịch vụ, Combo)
- **User Experience**: Giao diện thân thiện, dễ sử dụng cho cả admin và người dùng

## 🏥 HEALTHCARE DOMAIN ANALYSIS

### Pharmaceutical Industry Challenges
1. **Regulatory Compliance**: Tuân thủ quy định về dược phẩm
2. **Inventory Management**: Quản lý hạn sử dụng, lô sản xuất
3. **Cross-category Integration**: Kết hợp thuốc với dịch vụ y tế
4. **Real-time Tracking**: Theo dõi tồn kho và báo cáo tức thì

### KiotViet-Inspired Solutions
- **4-Category System**: Medicine, Goods, Services, Combo
- **Advanced UI/UX**: Tab navigation, modal forms, wizard interfaces
- **Real-time Features**: Live inventory tracking và notifications
- **Cross-category Relationships**: Flexible product bundling

## 👥 USER PERSONAS & WORKFLOWS

### Primary Users

#### 1. **Pharmacy Admin** (Primary)
**Goals:**
- Quản lý tồn kho thuốc hiệu quả
- Tạo combo sản phẩm để tăng doanh thu
- Báo cáo chi tiết về doanh số và tồn kho
- Tuân thủ quy định về dược phẩm

**Workflows:**
```
1. Đăng nhập → Dashboard tổng quan
2. Quản lý sản phẩm → 4 tab: Thuốc, Hàng hóa, Dịch vụ, Combo
3. Tạo sản phẩm mới → Wizard form với validation
4. Tạo combo → Chọn sản phẩm từ các category
5. Quản lý tồn kho → Real-time tracking
6. Báo cáo → Analytics dashboard
```

#### 2. **Healthcare Staff** (Secondary)
**Goals:**
- Tìm kiếm thuốc nhanh chóng
- Tạo đơn thuốc với combo dịch vụ
- Cập nhật thông tin bệnh nhân

**Workflows:**
```
1. Tìm kiếm thuốc → Advanced search & filter
2. Tạo đơn thuốc → Medicine + Service combo
3. Cập nhật tồn kho → Real-time inventory
4. Báo cáo → Patient care analytics
```

#### 3. **End Users** (Tertiary)
**Goals:**
- Tìm kiếm thông tin thuốc
- Đặt lịch dịch vụ y tế
- Xem combo sản phẩm

**Workflows:**
```
1. Tìm kiếm → Product catalog
2. Xem chi tiết → Product information
3. Đặt hàng → Shopping cart
4. Thanh toán → Payment processing
```

## 🔄 CORE WORKFLOWS

### Product Management Workflow (KiotViet-Inspired)

#### 1. **Product Creation Process**
```
Step 1: Choose Category
├── Medicine (Thuốc)
├── Goods (Hàng hóa)
├── Services (Dịch vụ)
└── Combo (Đóng gói)

Step 2: Fill Basic Information
├── Product name, code, description
├── Category assignment
├── Pricing information
└── Inventory settings

Step 3: Category-specific Details
├── Medicine: Active ingredients, dosage, manufacturer
├── Goods: Specifications, brand, supplier
├── Services: Duration, service type, requirements
└── Combo: Component selection, pricing rules

Step 4: Advanced Settings
├── Image upload
├── SEO settings
├── Related products
└── Cross-category relationships
```

#### 2. **Combo Management Workflow**
```
Step 1: Combo Planning
├── Define combo purpose
├── Select target audience
├── Set pricing strategy
└── Plan inventory requirements

Step 2: Product Selection
├── Browse medicine catalog
├── Add healthcare services
├── Include general goods
└── Set quantities và pricing

Step 3: Combo Configuration
├── Set combo pricing rules
├── Define discount structure
├── Configure inventory tracking
└── Set expiration policies

Step 4: Launch & Monitor
├── Activate combo
├── Monitor sales performance
├── Track inventory levels
└── Adjust pricing strategy
```

#### 3. **Inventory Management Workflow**
```
Real-time Tracking:
├── Live inventory updates
├── Low stock alerts
├── Expiry date notifications
└── Automatic reorder suggestions

Reporting & Analytics:
├── Sales performance
├── Inventory turnover
├── Profit margin analysis
└── Category performance
```
